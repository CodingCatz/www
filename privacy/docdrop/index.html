DocDrop 隱私權政策

最後更新：2026-06-20

DocDrop（以下稱「本擴充」）是一個在瀏覽器中閱讀 Markdown / JSON / Mermaid / 原始碼檔案的閱讀器。本政策說明本擴充如何處理您的資料。

1. 資料蒐集

本擴充不蒐集、不儲存、不傳送任何個人資料。

2. 檔案處理

您開啟或拖入的檔案，完全在您的瀏覽器本機解析與顯示，不會上傳到任何伺服器，也不會離開您的裝置。

3. 權限與網路

本擴充未要求任何特殊權限，運作時不進行任何網路請求，不含追蹤、分析或廣告。

4. 資料刪除

本擴充不保留任何資料；關閉分頁即結束。您可透過 chrome://extensions 移除本擴充。

5. 聯絡方式

如有任何疑問，請來信：catz.software@gmail.com

6. 政策變更

本政策若有更新，將於本頁公布並更新上方日期。

DocDrop Privacy Policy

Last updated: 2026-06-20

DocDrop ("the Extension") is an in-browser reader for Markdown / JSON / Mermaid / source-code files. This policy explains how the Extension handles your data.

1. Data collection

The Extension does not collect, store, or transmit any personal data.

2. File handling

Files you open or drop are parsed and displayed entirely locally in your browser. They are never uploaded to any server and never leave your device.

3. Permissions and network

The Extension requests no special permissions and makes no network requests. It contains no tracking, analytics, or ads.

4. Data deletion

The Extension retains no data; closing the tab ends the session. You can remove it via chrome://extensions.

5. Contact

Questions: catz.software@gmail.com

6. Changes

If this policy is updated, the changes will be posted on this page with an updated date above.
