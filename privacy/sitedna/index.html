SiteDNA 隱私權政策

最後更新：2026-06-27

SiteDNA（以下稱「本擴充」）是一個分析網頁設計與技術的瀏覽器擴充，會將當前網頁的技術棧、設計 token、元件、圖片與版面結構整理成報告。本政策說明本擴充如何處理您的資料。

1. 資料蒐集

本擴充不蒐集、不遠端儲存、不傳送任何個人資料，亦無使用者帳號或追蹤。

2. 網頁分析處理

當您主動點擊「分析」時，本擴充才會讀取當前分頁的 DOM 與計算後樣式以產生報告。所有分析完全在您的瀏覽器本機進行，網頁內容不會被上傳，也不會離開您的裝置。

3. 本機儲存

分析結果暫存於 chrome.storage.session（瀏覽器工作階段結束即自動清除）以傳遞給結果分頁；您的語言偏好存於 chrome.storage.local。兩者皆僅存在您的裝置本機，不外傳。

4. 權限說明

本擴充採最小權限：activeTab 與 scripting（僅在您點擊分析時，對您選定的分頁注入分析程式）、storage（暫存結果與語言偏好）、downloads（供您下載報告）。本擴充未要求任何網站主機權限，未經您操作不會接觸任何分頁。

5. 網路與第三方

本擴充本身不發出任何網路請求，不含追蹤、分析或廣告。選用的「送到 DocDrop」功能僅在您點擊時，透過瀏覽器本機的擴充間訊息把報告交給 DocDrop 擴充，不經由網際網路傳輸。

6. 資料刪除

工作階段資料會自動清除；您可透過 chrome://extensions 移除本擴充以清除語言偏好等本機資料。

7. 聯絡方式

如有任何疑問，請來信：catz.software@gmail.com

8. 政策變更

本政策若有更新，將於本頁公布並更新上方日期。

SiteDNA Privacy Policy

Last updated: 2026-06-27

SiteDNA ("the Extension") is a browser extension that analyzes a web page's design and technology, turning the current page's tech stack, design tokens, components, images and layout into a report. This policy explains how the Extension handles your data.

1. Data collection

The Extension does not collect, store remotely, or transmit any personal data. There are no accounts and no tracking.

2. Page analysis

Only when you actively click "Analyze" does the Extension read the current tab's DOM and computed styles to build the report. All analysis runs entirely locally in your browser; page content is never uploaded and never leaves your device.

3. Local storage

The analysis result is held temporarily in chrome.storage.session (cleared automatically when the browser session ends) to pass it to the results tab; your language preference is kept in chrome.storage.local. Both stay on your device only and are never sent elsewhere.

4. Permissions

The Extension uses minimal permissions: activeTab and scripting (inject the analyzer into the tab you choose, only when you click Analyze), storage (hold the result and language preference), and downloads (let you save the report). It requests no host permissions and never touches a tab without your action.

5. Network and third parties

The Extension itself makes no network requests and contains no tracking, analytics, or ads. The optional "Send to DocDrop" feature, only when you click it, passes the report to the DocDrop extension via a local in-browser message and does not transmit data over the internet.

6. Data deletion

Session data clears automatically; you can remove the Extension via chrome://extensions to clear local data such as the language preference.
